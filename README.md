# Lingshi Platform

去中心化多 Agent 自组织生态系统（规则引擎 + 竞技场）。

## 目标

- 无中心编排：平台定义规则，不指定执行者
- 灵石激励驱动：竞争、协作、晋升、降级
- 本地优先：单机可运行，后续可扩展多节点
- 可视化控制台：实时观察任务池、排行榜、生态健康

## 当前阶段

- Status: `Kickoff`
- Milestone: `MVP-10D`

## 计划中的技术栈

- Backend: Node.js + Fastify + TypeScript
- DB: SQLite（MVP）
- Realtime: WebSocket
- Dashboard: React + Vite + ECharts

## 快速开始（施工中）

```bash
npm run dev
```
